## Opções recomendadas para o perfil do produto

### Frontend (Web — Dashboard gestor)
[ ] Next.js 14+ com TypeScript (recomendado: App Router + Server Components)
[ ] Tailwind CSS para estilização
[ ] shadcn/ui para componentes

### Frontend (Mobile — Campo: engenheiro e funcionário)
[ ] Flutter (Dart) — app único para iOS e Android
[ ] React Native — alternativa se time já conhece JS/TS

### Backend / API
[ ] Node.js com Fastify ou Express (TypeScript)
[ ] Python com FastAPI — alternativa

### Banco de Dados
[ ] PostgreSQL (recomendado para dados relacionais + RLS para permissões)
[ ] Supabase — alternativa gerenciada com PostgreSQL + auth embutido

### Autenticação
[ ] JWT com refresh token rotation
[ ] Supabase Auth — se usar Supabase
[ ] NextAuth.js — se frontend Next.js

### Hospedagem / Infraestrutura
[ ] Railway ou Render (backend — simples e barato para MVP)
[ ] Vercel (frontend Next.js)
[ ] Supabase (banco + auth gerenciado)
